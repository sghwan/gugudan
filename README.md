# gugudan

## 목표
* 개인 프로젝트 프로세스 경험

## 첫번째 요구 사항
* 2, 3단 출력
* System.out.println()을 사용

## 두번째 요구 사항
* 입력 받은 값을 통한 구구단 출력

## 세번째 요구 사항
* 입력 받은 값에 대한 구구단 출력
* 단, 2~9 외의 숫자가 입력 되면 처리 불가한 메세지 출력

## 네번째 요구 사항
* 배열을 사용한 구구단 

## 다섯번째 요구 사항
* 메소드를 활용한 구구단 출력

## 여섯번째 요구 사항
* 클래스 도입
* 프로그램 시작점과 기능 역할을 분리
